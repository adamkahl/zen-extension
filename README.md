# Zen Tidy Up

A Firefox extension to organize and rename browser tabs based on URL patterns.

## Features

- Create custom URL patterns to rename tabs
- Organize tabs into groups with categories
- Color-code tabs with emoji indicators
- Drag-and-drop group reordering
- Import/Export settings

## Usage

1. Click the extension icon to organize your tabs
2. Open settings to configure URL patterns and groups
3. Tabs will be automatically renamed and sorted by group

## Privacy

This extension does not collect or transmit any data. All settings are stored locally in your browser.
